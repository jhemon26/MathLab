Q1(ক) Pair of Straight Lines
QUESTION
Show that $x^2-5xy+4y^2+x+2y-2=0$ represents a pair of straight lines. Find the point of intersection and the angle between them.
ANSWER
The two lines are: x − y − 1 = 0 and x − 4y + 2 = 0. Intersection point: (2, 1). Angle between them: arctan(3/5) ≈ 30.96°
CODE
Mathematica
(* Verify degeneracy & factor into two lines *)
eq = x^2 - 5 x y + 4 y^2 + x + 2 y - 2;
Factor[eq]

(* Solve intersection of the two lines *)
sol = Solve[{x - y - 1 == 0, x - 4 y + 2 == 0}, {x, y}]

(* Angle between lines *)
m1 = 1; m2 = 1/4;
angle = ArcTan[Abs[(m1 - m2)/(1 + m1*m2)]] // N

(* Plot the pair of lines *)
ContourPlot[eq == 0, {x, -2, 6}, {y, -2, 4}, PlotPoints -> 100]
EXPLANATION

For the general second-degree equation $ax^2+2hxy+by^2+2gx+2fy+c=0$, the coefficients here are $a=1,\ h=-5/2,\ b=4,\ g=1/2,\ f=1,\ c=-2$. A pair-of-lines condition requires the determinant $\Delta = abc+2fgh-af^2-bg^2-ch^2 = 0$. Substituting: $(1)(4)(-2)+2(1)(1/2)(-5/2)-(1)(1)-(4)(1/4)-(-2)(25/4) = -8-2.5-1-1+12.5 = 0$. Confirmed.

Since $x^2-5xy+4y^2$ factors as $(x-y)(x-4y)$, the full equation is $(x-y+l)(x-4y+m)=0$. Matching the linear and constant terms gives $l=-1,\ m=2$.

Solving the two line equations simultaneously gives the intersection point $(2,1)$. The angle formula $\tan\theta=|\frac{m_1-m_2}{1+m_1 m_2}|$ with slopes $1$ and $1/4$ gives $\theta=\arctan(3/5)\approx30.96°$.

Q1(খ) Identify & Reduce a Conic
QUESTION
Identify the conic $x^2+12xy-4y^2-6x+4y+9=0$, reduce it to standard form, and plot it.
ANSWER
Type: Hyperbola. Center: (0, 0.5). Standard form (rotated axes): $$\frac{Y^2}{5/4} - \frac{X^2}{2} = 1$$
CODE
Mathematica
(* Find center, eigenvalues for rotation, and plot *)
eq = x^2 + 12 x y - 4 y^2 - 6 x + 4 y + 9;

center = Solve[{D[eq, x] == 0, D[eq, y] == 0}, {x, y}]

mat = {{1, 6}, {6, -4}};
{evals, evecs} = Eigensystem[mat]

ContourPlot[eq == 0, {x, -4, 4}, {y, -4, 4},
  PlotPoints -> 150]
EXPLANATION

Here $h^2-ab=36-(1)(-4)=40>0$, classifying this as a hyperbola. The center is found by setting both partial derivatives to zero and solving: gives $(0, 0.5)$.

Shifting the origin to the center eliminates linear terms. The coefficient matrix $\begin{pmatrix}1&6\\6&-4\end{pmatrix}$ has eigenvalues $5$ and $-8$ — these become the new quadratic coefficients after rotating axes to align with the eigenvectors. The resulting equation $5X^2-8Y^2=-10$ rearranges to the standard form shown.

Q2(ক) Partial Derivatives
QUESTION
For $f(x,y)=x^2 e^{xy}$, find all first and second partial derivatives.
ANSWER
$$f_x = e^{xy}(2x+x^2y),\quad f_y = x^3 e^{xy}$$ $$f_{xx} = e^{xy}(2+4xy+x^2y^2),\quad f_{yy}=x^4 e^{xy},\quad f_{xy}=e^{xy}(3x^2+x^3y)$$
CODE
Mathematica
f[x_, y_] := x^2 Exp[x y]

fx  = D[f[x, y], x]        // Simplify
fy  = D[f[x, y], y]        // Simplify
fxx = D[f[x, y], {x, 2}]   // Simplify
fyy = D[f[x, y], {y, 2}]   // Simplify
fxy = D[f[x, y], x, y]    // Simplify
EXPLANATION

For $f_x$, use the product rule treating $y$ as constant: $\frac{\partial}{\partial x}(x^2)\cdot e^{xy} + x^2\cdot\frac{\partial}{\partial x}(e^{xy}) = 2xe^{xy}+x^2ye^{xy}$. For $f_y$, $x$ is constant, so only $e^{xy}$ differentiates: $x^2\cdot xe^{xy}=x^3e^{xy}$.

Second derivatives repeat the same process on each first derivative. Clairaut's theorem guarantees $f_{xy}=f_{yx}$ since $f$ is smooth everywhere.

Q2(খ) Continuity at Origin
QUESTION
Show that $f(x,y)=\frac{xy(x^2-y^2)}{x^2+y^2}$ for $(x,y)\neq(0,0)$, $f(0,0)=0$, is continuous at the origin.
ANSWER
The limit as $(x,y)\to(0,0)$ equals $0 = f(0,0)$ — continuous at the origin.
CODE
Mathematica
f[x_, y_] := (x y (x^2 - y^2))/(x^2 + y^2);

(* Polar substitution to show bound *)
polar = f[r Cos[t], r Sin[t]] // Simplify
Limit[polar, r -> 0]
EXPLANATION

Switching to polar coordinates $x=r\cos\theta,\ y=r\sin\theta$ transforms the function into $r^2\cos\theta\sin\theta(\cos^2\theta-\sin^2\theta)$. The angular factor is bounded between $-1$ and $1$ regardless of $\theta$.

So $|f|\le r^2$, and as $r\to0$ from any direction, $r^2\to0$ forces the function to $0$, matching $f(0,0)=0$. This direction-independence is exactly what continuity requires.

Q3 Three Initial Value Problems
QUESTION
Solve and plot:
(i) $y''-7y'+10y=0,\ y(0)=0,\ y'(0)=3$
(ii) $y''+4y=2\sin2x,\ y(0)=0,\ y'(0)=0$
(iii) $(x^2-y^2)dx-xy\,dy=0,\ y(1)=1$
ANSWER
(i) $y=e^{5x}-e^{2x}$    (ii) $y=\frac{1}{4}\sin2x-\frac{x}{2}\cos2x$    (iii) $x^4-2x^2y^2=-1$
CODE
Mathematica
(* (i) Constant-coefficient homogeneous ODE *)
sol1 = DSolve[{y''[x] - 7 y'[x] + 10 y[x] == 0,
  y[0] == 0, y'[0] == 3}, y[x], x]
Plot[y[x] /. sol1, {x, 0, 1}]

(* (ii) Resonant forced oscillator *)
sol2 = DSolve[{y''[x] + 4 y[x] == 2 Sin[2 x],
  y[0] == 0, y'[0] == 0}, y[x], x]
Plot[y[x] /. sol2, {x, 0, 6}]

(* (iii) Homogeneous first-order ODE *)
sol3 = DSolve[{(x^2 - y[x]^2) - x y[x] y'[x] == 0,
  y[1] == 1}, y[x], x]
Plot[Evaluate[y[x] /. sol3], {x, 0.2, 3}]
EXPLANATION

(i) Characteristic equation $r^2-7r+10=0$ factors as $(r-2)(r-5)=0$, giving $y=C_1e^{2x}+C_2e^{5x}$. From $y(0)=0$: $C_1+C_2=0$. From $y'(0)=3$: $2C_1+5C_2=3$. Solving: $C_1=-1,\ C_2=1$.

(ii) The natural frequency $\omega=2$ matches the forcing $2\sin2x$ — resonance. The particular solution must take the form $y_p=x(A\cos2x+B\sin2x)$. Substituting into the ODE gives $A=-1/2,\ B=0$. Initial conditions fix the homogeneous part.

(iii) The ODE is homogeneous in $x,y$. Substituting $y=vx$ reduces it to a separable equation. Integrating and applying $y(1)=1$ yields the implicit solution $x^4-2x^2y^2=-1$.

Q4(ক) Bisection Method
QUESTION
Using the bisection method, find a root of $\cos x - 1.3x = 0$ correct to three decimal places.
ANSWER
Root ≈ 0.624
CODE
Mathematica
f[x_] := Cos[x] - 1.3 x;
a = 0; b = 1;
While[Abs[b - a] > 0.0005,
  c = (a + b)/2;
  If[f[a] f[c] < 0, b = c, a = c]
];
root = (a + b)/2 // N
EXPLANATION

$f(0)=1>0$ and $f(1)\approx-0.76<0$, so a root exists in $(0,1)$ by the Intermediate Value Theorem. Each iteration bisects the interval and keeps the half where the sign change persists. After about 11 iterations, the interval shrinks below $0.0005$, giving three-decimal accuracy.

Q4(খ) Simpson's 1/3 Rule
QUESTION
Using Simpson's 1/3 rule, write Mathematica code to evaluate $\displaystyle\int_0^1 e^{x^2}\,dx$.
ANSWER
Composite Simpson's $\frac{1}{3}$ rule with $n=10$: 1.4627.
Single-panel Simpson's $\frac{1}{3}$ rule: 1.4757.
CODE
Mathematica
f[x_] := Exp[x^2];
n = 10; a = 0; b = 1; h = (b - a)/n;
xs = Table[a + i*h, {i, 0, n}];
ys = f /@ xs;

integral = (h/3)*(ys[[1]] + ys[[-1]] +
     4*Total[ys[[Range[2, n, 2]]]] +
     2*Total[ys[[Range[3, n - 1, 2]]]]);

Print["Simpson's 1/3 = ", integral]
singlePanel = (b - a)/6 * (f[a] + 4f[(a + b)/2] + f[b]);
Print["Single-panel = ", singlePanel]
Print["NIntegrate   = ", NIntegrate[f[x], {x, 0, 1}]]
EXPLANATION

Simpson's 1/3 rule fits parabolas through consecutive triples of points. With $n=10$ (even required), the composite formula weights endpoints by 1, odd-indexed interior points by 4, and even-indexed interior points by 2, all scaled by $h/3$. If only one Simpson panel is expected, use $\frac{b-a}{6}[f(a)+4f((a+b)/2)+f(b)]$, which gives about $1.4757$ for this integral.

Q5(ক) Cramer's Rule
QUESTION
Using Cramer's rule, solve: $2x+y+z=7,\quad x-4y+3z=2,\quad 3x+2y+2z=13$
ANSWER
x = 1,   y = 2,   z = 3
CODE
Mathematica
A = {{2, 1, 1}, {1, -4, 3}, {3, 2, 2}};
b = {7, 2, 13};

D0 = Det[A];
Ax = A; Ax[[All, 1]] = b; Dx = Det[Ax];
Ay = A; Ay[[All, 2]] = b; Dy = Det[Ay];
Az = A; Az[[All, 3]] = b; Dz = Det[Az];

{Dx, Dy, Dz}/D0
EXPLANATION

Cramer's rule solves $Ax=b$ by replacing one column of $A$ at a time with $b$ and comparing determinants. Here $\det(A)=-7$, and replacing the 1st, 2nd, 3rd columns gives $D_x=-7,\ D_y=-14,\ D_z=-21$. Dividing each by $D_0=-7$ yields $x=1, y=2, z=3$.

Q5(খ) Right-angled Triangle from Vectors
QUESTION
Show that $\vec a=3\hat i-2\hat j+\hat k$, $\vec b=\hat i-3\hat j+5\hat k$, $\vec c=2\hat i+\hat j-4\hat k$ form the sides of a right-angled triangle.
ANSWER
$\vec a\cdot\vec c=0$ (perpendicular) and $|\vec a|^2+|\vec c|^2=14+21=35=|\vec b|^2$ — right triangle confirmed.
CODE
Mathematica
a = {3, -2, 1}; b = {1, -3, 5}; c = {2, 1, -4};

a - b == c               (* True — they close into a triangle *)
a.c                       (* 0 — perpendicular *)
{Norm[a]^2, Norm[c]^2, Norm[b]^2} (* {14, 21, 35} *)
Norm[a]^2 + Norm[c]^2 == Norm[b]^2  (* True *)
EXPLANATION

First, $\vec a-\vec b = \vec c$ confirms the three vectors close into a triangle when placed head-to-tail. The dot product $\vec a\cdot\vec c = 6-2-4 = 0$ proves sides $\vec a$ and $\vec c$ are perpendicular. Finally, $|\vec a|^2+|\vec c|^2 = 14+21 = 35 = |\vec b|^2$ satisfies the Pythagorean theorem, independently confirming the right angle with $\vec b$ as hypotenuse.

Q6(ক) Newton–Raphson Method
QUESTION
Using Newton–Raphson's method, find a root of $x^3-x-1=0$ correct to three decimal places.
ANSWER
Root ≈ 1.325
CODE
Fortran 90
program newton_raphson
    implicit none
    real    :: x0, x1, f, fd
    integer :: i

    x0 = 1.5
    do i = 1, 50
        f  = x0**3 - x0 - 1.0
        fd = 3.0*x0**2 - 1.0
        x1 = x0 - f/fd
        if (abs(x1 - x0) < 0.0005) exit
        x0 = x1
    end do

    print *, "Root = ", x1
end program newton_raphson
EXPLANATION

Starting from $x_0=1.5$, each step applies $x_{n+1}=x_n - f(x_n)/f'(x_n)$, where $f(x)=x^3-x-1$ and $f'(x)=3x^2-1$. Newton–Raphson converges quadratically near a simple root, so only a few iterations are needed to reach three-decimal accuracy.

Q6(খ) Fixed Point Iteration
QUESTION
Using fixed point iteration, find a real root of $x - \cos x = 0$.
ANSWER
Root ≈ 0.739 (the Dottie number)
CODE
Fortran 90
program fixed_point
    implicit none
    real    :: x0, x1
    integer :: i

    x0 = 0.5
    do i = 1, 100
        x1 = cos(x0)
        if (abs(x1 - x0) < 0.0005) exit
        x0 = x1
    end do

    print *, "Root = ", x1
end program fixed_point
EXPLANATION

Rearranging $x-\cos x=0$ as $x=\cos x$ turns root-finding into repeatedly applying $x_{n+1}=\cos(x_n)$. Since $|g'(x)|=|\sin x|<1$ near the root, the fixed-point condition holds and convergence is guaranteed.

Q7(ক) Weddle's Rule
QUESTION
Dividing into 6 parts, use Weddle's rule to evaluate $\displaystyle\int_a^b \frac{dx}{1+x^2}$.
ANSWER
Let $h=\frac{b-a}{6}$ and $x_i=a+ih$. Weddle's rule gives $$\int_a^b \frac{dx}{1+x^2}\approx \frac{3h}{10}\left[y_0+5y_1+y_2+6y_3+y_4+5y_5+y_6\right],\quad y_i=\frac{1}{1+x_i^2}.$$
CODE
Fortran 90
program weddle_rule
    implicit none
    real :: a, b, h, x(0:6), y(0:6), s, integral
    integer :: i

    print *, "Enter lower limit a and upper limit b:"
    read *, a, b
    h = (b - a) / 6.0

    do i = 0, 6
        x(i) = a + i*h
        y(i) = 1.0 / (1.0 + x(i)**2)
    end do

    s = y(0) + 5.0*y(1) + y(2) + 6.0*y(3) + y(4) + 5.0*y(5) + y(6)
    integral = (3.0*h/10.0) * s

    print *, "Integral = ", integral
end program weddle_rule
EXPLANATION

Weddle's rule requires exactly 6 subintervals (7 points). Since the paper gives the limits as $a$ and $b$, the correct answer should stay in formula form or let the program read $a$ and $b$ as input.

Q7(খ) Matrix Addition & Subtraction
QUESTION
Write a Fortran program to add and subtract two square matrices.
ANSWER
General program — works for any $n \times n$ matrices.
CODE
Fortran 90
program matrix_add_sub
    implicit none
    integer, parameter :: n = 3
    real :: A(n,n), B(n,n), C_add(n,n), C_sub(n,n)
    integer :: i, j

    print *, "Enter elements of A (row by row):"
    read *, ((A(i,j), j=1,n), i=1,n)
    print *, "Enter elements of B (row by row):"
    read *, ((B(i,j), j=1,n), i=1,n)

    do i = 1, n
        do j = 1, n
            C_add(i,j) = A(i,j) + B(i,j)
            C_sub(i,j) = A(i,j) - B(i,j)
        end do
    end do

    print *, "A + B ="
    do i = 1, n
        print *, (C_add(i,j), j=1,n)
    end do
    print *, "A - B ="
    do i = 1, n
        print *, (C_sub(i,j), j=1,n)
    end do
end program matrix_add_sub
EXPLANATION

Matrix addition and subtraction are element-wise: a nested loop computes $C_{add}(i,j)=A(i,j)+B(i,j)$ and $C_{sub}(i,j)=A(i,j)-B(i,j)$. The size $n=3$ is a parameter easily changed.

Q8 Gauss–Jacobi Iteration
QUESTION
Using the Gauss–Jacobi method, solve to 3 decimal places: $27x+6y-z=85,\quad 6x+15y+2z=72,\quad x+y+54z=110$
ANSWER
x ≈ 2.425,   y ≈ 3.573,   z ≈ 1.926
CODE
Fortran 90
program gauss_jacobi
    implicit none
    real :: x, y, z, xn, yn, zn
    integer :: i

    x = 0.0; y = 0.0; z = 0.0

    do i = 1, 100
        xn = (85.0 - 6.0*y + z)  / 27.0
        yn = (72.0 - 6.0*x - 2.0*z) / 15.0
        zn = (110.0 - x - y)       / 54.0

        if (abs(xn-x) < 0.0005 .and. abs(yn-y) < 0.0005 &
            .and. abs(zn-z) < 0.0005) then
            x = xn; y = yn; z = zn
            exit
        end if
        x = xn; y = yn; z = zn
    end do

    print *, "x =", x, " y =", y, " z =", z
end program gauss_jacobi
EXPLANATION

Each equation is rearranged to isolate one variable, and all three are updated simultaneously from the previous iteration's values (Jacobi's distinguishing feature vs Gauss-Seidel). The system is diagonally dominant (27>6+1, 15>6+2, 54>1+1), guaranteeing convergence.

Q9 Euler's Method
QUESTION
Using Euler's method, solve $\frac{dy}{dx}=x+y$, $x=0$, $y=0$, from $x=0$ to $x=1.0$ with $h=0.2$.
ANSWER
$y(1.0) \approx$ 0.4883
x0.00.20.40.60.81.0
y000.040.1280.27360.4883
CODE
Fortran 90
program euler_method
    implicit none
    real    :: x, y, h
    integer :: i, n

    x = 0.0; y = 0.0; h = 0.2
    n = 5   ! (1.0 - 0.0) / 0.2

    do i = 1, n
        y = y + h * (x + y)
        x = x + h
        print *, "x =", x, "  y =", y
    end do
end program euler_method
EXPLANATION

Euler's method steps forward using the slope at the current point: $y_{n+1}=y_n+h\cdot f(x_n,y_n)$, where $f(x,y)=x+y$. With $h=0.2$, five steps carry the solution from $x=0$ to $x=1.0$. Each new $y$ depends only on the immediately preceding value.

Q10(ক) Largest of Ten Numbers
QUESTION
Write a Fortran program to find the largest number among ten given numbers.
ANSWER
Single-pass comparison finds the maximum.
CODE
Fortran 90
program largest_number
    implicit none
    real    :: num(10), largest
    integer :: i

    print *, "Enter 10 numbers:"
    read *, (num(i), i=1,10)

    largest = num(1)
    do i = 2, 10
        if (num(i) > largest) largest = num(i)
    end do

    print *, "Largest = ", largest
end program largest_number
EXPLANATION

The first element is taken as the initial guess for the maximum. Each subsequent element is compared — whenever a larger value appears, it replaces the current largest. After one pass through all ten numbers, the true maximum remains.

Q10(খ) Matrix Multiplication (3×4 by 4×5)
QUESTION
Given $A$ is $3\times4$ and $B$ is $4\times5$, write a Fortran program to compute $C=AB$.
ANSWER
Result $C$ is a $3\times5$ matrix computed via the standard triple loop.
CODE
Fortran 90
program matrix_multiply
    implicit none
    integer, parameter :: m = 3, n = 4, p = 5
    real :: A(m,n), B(n,p), C(m,p)
    integer :: i, j, k

    print *, "Enter A (3x4):"
    read *, ((A(i,j), j=1,n), i=1,m)
    print *, "Enter B (4x5):"
    read *, ((B(i,j), j=1,p), i=1,n)

    do i = 1, m
        do j = 1, p
            C(i,j) = 0.0
            do k = 1, n
                C(i,j) = C(i,j) + A(i,k) * B(k,j)
            end do
        end do
    end do

    print *, "C = A x B ="
    do i = 1, m
        print *, (C(i,j), j=1,p)
    end do
end program matrix_multiply
EXPLANATION

Matrix multiplication is valid since $A$'s column count (4) matches $B$'s row count (4). The result is $3\times5$. Each entry $C(i,j)$ is the dot product of row $i$ of $A$ with column $j$ of $B$, computed by the innermost loop accumulating $A(i,k)\times B(k,j)$.
